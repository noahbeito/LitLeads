## Installation and usage
```
npm install
npm run build
npm run start
```
## APIs
- [Google Books](https://developers.google.com/books)
- [Forismatic](https://forismatic.com/en/api/)
- [Philosophy Quotes API](https://github.com/KaranDahiya/philosophy-quotes-API)
